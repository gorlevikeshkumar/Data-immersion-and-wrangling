# Data-immersion-and-wrangling
 ## Objective
This project is about understanding and cleaning data using Python. The main goal is to convert raw data into a clean and organized format for better analysis.
## Tools Used
Python
Jupyter Notebook
Pandas
NumPy
Matplotlib
## Working
Imported the dataset, handled missing values and duplicates, cleaned the data, and performed basic analysis and visualization to make the dataset ready for further use
<br>

1. Import libraries

4. Check missing values
5. Remove duplicate records

7. Save cleaned dataset
